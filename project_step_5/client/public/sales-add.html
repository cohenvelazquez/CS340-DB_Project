Sale Recording Process

  • Select existing customer or create new customer first
  • Choose the appropriate event where sale occurred
  • Enter exact date and time of transaction
  • Total amount should match sum of all items sold
  • After recording sale, add individual items via SoldItems

Next Steps After Sale Creation

After creating this sale record, you'll need to:

  1. Go to the Sold Items page
  2. Add each individual item purchased in this sale
  3. Update item status to "Sold" in the Items table
  4. Verify total amounts match
